# Security Policy

## Supported versions

No version is published or supported yet. The 0.1.x line is intended to receive
security fixes after the repository and its private reporting route are live.

## Reporting a vulnerability

Repository creation and enabling GitHub private vulnerability reporting are
pre-tag release blockers. Until both exist, there is no published reporting
route; do not disclose exploit details in a public issue. After publication,
use GitHub private vulnerability reporting for the AdaptiveGlyphKit repository.

The maintainer will acknowledge reports received through that private route,
investigate them, and coordinate disclosure when appropriate. This
acknowledgement is not a response-time or resolution-time guarantee.
