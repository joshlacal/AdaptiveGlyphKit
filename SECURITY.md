# Security Policy

## Supported versions

The latest 0.1.x release receives security fixes.

## Reporting a vulnerability

Use [GitHub private vulnerability reporting](https://github.com/joshlacal/AdaptiveGlyphKit/security/advisories/new)
for the AdaptiveGlyphKit repository. Do not disclose exploit details in a
public issue.

The maintainer will acknowledge reports received through that private route,
investigate them, and coordinate disclosure when appropriate. This
acknowledgement is not a response-time or resolution-time guarantee.
